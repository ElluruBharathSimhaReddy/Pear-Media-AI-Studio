# Pear Media AI Studio

Pear Media AI Studio is a responsive React-based prototype built for the Pear Media assignment. It connects simple user input with AI-inspired workflows through a clean and modern interface.

## Features

### Workflow A, Text to Image
- Enter a simple text prompt
- Enhance the prompt into a richer visual description
- Human-in-the-loop review flow
- Generate image placeholder for demo

### Workflow B, Image to Variations
- Upload a reference image
- Preview the selected image
- Show mock analysis details such as subject, style, lighting, and color palette
- Variation generation placeholder for demo
